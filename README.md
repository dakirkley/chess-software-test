# Chess for Beginners

An interactive chess game designed for young children learning to play chess.

## Features

- ♟️ Full chess board with drag-and-drop pieces
- 💡 Visual hints showing valid moves
- ✅ Move validation using chess.js
- ↩️ Undo last move
- 🔄 Reset game
- 📱 Responsive design for tablets and desktops
- 🎨 Kid-friendly colorful UI

## Tech Stack

- Next.js 14
- React + TypeScript
- Tailwind CSS
- chess.js
- react-chessboard

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play!

## Deployment

Built for static export to Vercel:

```bash
npm run build
```

Output is in `dist/` folder.
